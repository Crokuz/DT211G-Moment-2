# Moment 2, DT211G

Detta projektet är utvecklat i samband med kursen "Frontend-baserad webbutveckling” vid ­Mittuniversitetet i Sundsvall. Syftet är att ge kursdeltagaren färdigheter inom användningen av FetchAPI-anrop, sortering och filtrering av inmatad data samt praktiska erfarenheter inom automatisering av en utvecklingsmiljö, versionshantering och publicering.

Projektet finns publicerat på [Netlify](https://leno2003-dt211g-moment2.netlify.app/).
